## ✨ Fitur Utama

- 📍 **Check-in/Check-out** — Pencatatan waktu datang dan pulang secara akurat.
- 📁 **Local Storage** — Database berbasis JSON yang ringan tanpa perlu SQL.
- 📊 **Tabel Laporan** — Riwayat kehadiran tertata rapi di terminal.
- 🎨 **UI/UX Modern** — Antarmuka web menggunakan HTML5 & Vanilla CSS.
- 🔒 **Session Security** — Menggunakan Express-Session untuk keamanan data.

---

## 🚀 Cara Penginstallan

Ikuti langkah langkah berikut untuk setup di perangkat kamu:

### 1️⃣ Inisialisasi Project
```bash
npm init -y
```

### 2️⃣ Install Dependensi
```bash
npm install express express-session
```

### 3️⃣ Jalankan Server
```bash
node server.js
```

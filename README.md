# 📝 Terminal Absence System
# Sistem absensi berbasis Command Line Interface (CLI) yang simpel dan cepat. Dibuat menggunakan Node.js untuk memudahkan pencatatan kehadiran langsung melalui terminal favorit kamu.

✨ Fitur Utama 
Check-in/Check-out: Catat jam datang dan pulang dengan satu perintah.
Local Storage: Data disimpan secara aman dalam format JSON lokal.
Laporan Harian: Lihat riwayat absensi kamu dalam bentuk tabel di terminal.
Validasi Waktu: Mencegah absensi ganda di hari yang sama.
Tampilan Ui/Ux: html5 dan vanila style css

cara penginstallan 
1. npm install
2. npm init -y
3. npm install express express-session
4. node server.js
